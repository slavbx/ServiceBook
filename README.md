# ServiceBook
Веб-приложение на Spring MVC + PostgreSQL + Liquibase для ведения электронной сервисной книги по личному автомобилю.
Основное назначение - контроль интервалов обслуживания и фиксация соответствующих работ.

### Информация  
Версия Java: 17  
Версия Maven: 4.0.0
